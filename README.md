# Pecunia

Plataforma de gestión de finanzas personales para registrar ingresos y egresos, categorizar gastos y visualizar informes.

## Tecnologías

- **Frontend:** Vue 3 (Composition API), TypeScript, Tailwind CSS, Pinia
- **Backend:** Node.js, Express, TypeScript

## Estructura del proyecto

- `client/` — Aplicación frontend (Vue)
- `server/` — API REST (Express)
